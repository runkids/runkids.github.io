VSCode無法更新最新版本

發表於 2017-12-27

今天要更新VSCode,結果跳出下面的訊息,然後就一直更新失敗！

Could not create temporary directory ：權限被拒絕

解決辦法：必須更改VSCode目錄的權限，在終端機輸入：

  • sudo chown -R $USER:admin ~/Library/Caches/com.microsoft.VSCode
  • sudo chown -R $USER:admin ~/Library/Caches/com.microsoft.VSCode.ShipIt
