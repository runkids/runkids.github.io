實做：使用JS控制CSS來改變圖片效果
學習重點
  • 建立 CSS 全域變數 :root （IE 和 Edge 目前不支援）
  • 利用 JS 動態改變 CSS 變數
解說
  • 定義CSS全域變數 :root{ --變數名稱 : 變數值 }
  • 使用變數 var( --變數名稱 )
  • 使用data attribute 定義單位 ex. px ，如果值是顏色就不需要
  • 使用 JS 取得 data attribute 值，即我們定義的單位
  • 使用 JS 動態改變 CSS 變數值

利用 document.documentElement.style.setProperty( '--變數名稱', '變數值' ) 動態改變 CSS，並且加上單位
